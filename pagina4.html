Áreas de Estudo em Engenharia de Dados.

  • Big Data e Sistemas Distribuídos:
    • Hadoop: Entenda o ecossistema Hadoop, incluindo HDFS e MapReduce.
    • Spark: Aprenda a processar grandes volumes de dados com Spark e explore bibliotecas como SparkSQL e MLlib.
  • Bancos de Dados NoSQL:
    • Cassandra: Um banco de dados distribuído popular para escalabilidade horizontal.
    • MongoDB: Um banco de dados orientado a documentos com flexibilidade de esquema.
    • Elasticsearch: Útil para buscas e análises em tempo real.
  • Processamento de Streams:
    • Kafka: Uma plataforma de streaming distribuída.
    • Apache Flink: Processamento em tempo real e análise de dados.
  • Ferramentas de Orquestração:
    • Apache Airflow: Para criar, agendar e monitorar fluxos de trabalho.
    • Apache NiFi: Para coletar, transformar e enviar dados.
  • Data Lakes e Armazenamento em Nuvem:
    • Amazon S3: Serviço de armazenamento de objetos em nuvem.
    • Google Cloud Storage e Azure Blob Storage: Outras opções populares para armazenamento em nuvem.
  • Infraestrutura como código (IAC):
    • Terraform: Para provisionar e gerenciar infraestrutura como código.
    • Ansible: Para automação e gerenciamento de configuração.
